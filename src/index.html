Instructions

The program randomly assigns students to rooms and generates correspoding exam rosters for each room as .csv and .pdf files. You can optionally upload a single page .pdf with exam instructions which will be appended as the second page of each roster pdf [This feature desn't work, will try to fix].

Please upload your student and room information by downloading and modifying the templates: students.csv and rooms.csv. You can export your student lists via MyBanner -> Class List -> Subject (for corresponding CRNs) -> Export -> Excel Spreadsheet .xls.

Even though I tested possible cases, please make sure that the total student count and the total room capacity are exactly the same and double check if all the students have been uniquely assigned to rooms in the generated files. I kindly ask you to notify me about issues if you encounter them.

You can and run the program locally. After downloading, make sure to unzip the folder and open index.html with any internet browser.
